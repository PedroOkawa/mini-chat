# Mini-Chat

This is a test using Android Architecture features based on a chat window
(Not a functional chat, it's just the layout and interactions with current messages)

# Features

* Room (Database)
* Dagger 2 (Dependency Injection)
* Databinding (View Injection)
* Retrofit 2 (Rest Interface)
* LiveData
* LifeCycle
* Glide (Image cache)
* Mockito (Mocking Framework)
* Espresso (Instrumentation Test)
